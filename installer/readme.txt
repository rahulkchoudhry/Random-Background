								Random Background
								-----------------
----------------------------------------------------------------------------------------------------------------------------------------------------

High quality desktop backgrounds
This software application will automatically set your desktop background to a high quality wallpaper when run.

----------------------------------------------------------------------------------------------------------------------------------------------------

random_background.exe : This executable is the main file which will set your background for you.

photo_information.exe : Information about your current background including the photographer and a download link can be accessed via this executable.

scheduled_changer.ps1 : This powershell script can be used to automatically schedule the random_background.exe to run regularly.
			Alternatively use Windows Task Scheduler to run the random_background.exe as per your preference.
